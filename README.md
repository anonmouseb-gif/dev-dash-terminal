# 💻 Dev-Dash Terminal
A custom, terminal-style dashboard built with Next.js and React. 

## 🚀 Features
- **Linux-Style UI:** A clean, dark-mode terminal interface.
- **Responsive Design:** Works on all screen sizes.
- **Built for Developers:** A minimalist hub for tracking projects.

## 🛠️ Tech Stack
- **Framework:** Next.js
- **Styling:** CSS-in-JS
- **Language:** TypeScript
